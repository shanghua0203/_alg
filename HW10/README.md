黎曼積分（切豆腐）：適合 低維度 (1D, 2D, 3D)。它很精確，像是一個強迫症的會計師，算得清清楚楚。但在高維度世界裡，它的工作量會大到把它壓垮。

蒙地卡羅（撒豆子）：適合 高維度 (4D 以上)。它不在乎維度有多高，因為它只是隨機抽樣。雖然每次算出來的答案都會「跳動」（有一點誤差），但它是高維度世界裡唯一跑得動的方法。
